# Gate 1 System
## Editor Training Manual
### Version 1.0

---

## 1. Purpose

This manual is written ONLY for Editors using the Gate 1 System.

Your role is critical. You are the first human gate in preserving evidence, quality, and accuracy.

---

## 2. Your Role as an Editor

You are responsible for:
- Handling raw interview videos
- Renaming videos manually and correctly
- Reporting quality issues immediately
- Backing up completed videos

You are NOT responsible for:
- Searching all videos
- Fixing camera teams
- Managing people or roles

---

## 3. Daily Editor Workflow (MANDATORY)

Follow this order strictly:

1. **Insert the SD card** into your computer
2. **Wait** for the Gate 1 Agent to detect the SD
3. **Confirm** camera number / SD label if prompted
4. **Copy** video files to your working folder
5. **Manually rename** each file following the format
6. **Observe** system guidance (toasts/modals)
7. **Correct** mistakes if guided
8. **Report** any quality issues immediately
9. **Back up** finished videos to registered disk
10. **Confirm** backup verification before finishing

---

## 5. Understanding Agent Guidance

### Green Toast (Success)
- File synced correctly
- No action needed

### Yellow Toast (Warning)
- Minor issue detected
- Review and fix if possible

### Red Modal (Error)
- Critical information missing
- Must be corrected before proceeding

## 7. SD Card Copy Confirmation

The Agent tracks your copy progress:

### Before Removing SD Card
- Check the copy status panel
- Ensure all files show "Copied"
- Wait for "Safe to Remove" confirmation

### If You Remove Early
- Warning will appear
- Group leader will be notified
- Decision is logged

**Never assume copying is complete without verification.**
